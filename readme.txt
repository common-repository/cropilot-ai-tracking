== Installation ==
1. Install the plugin through the WordPress plugins screen directly, or upload the plugin files to the `/wp-content/plugins/cropilot-tracking` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Once CroPilot.ai launches, you'll be able to sign up for an account and get your unique Client ID.
4. Enter your Client ID in the plugin settings to start tracking and receiving AI-powered insights.
